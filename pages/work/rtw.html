Through this course I learnt how to build a meaningful real-time web application. Socket.io was the technique I used to setup an open connection between the client and the server. With this I was able to send data in real-time both ways, at the same time. I created a chat-like application where users can come together in a room and share their favorite music tracks. After a user shared his track, other users can up or down vote this track. In each room a ranking is created of the shared tracks with the score behind the title.

This application is just a prototype. There are plenty of things that needs to be added, like a feature where users can create there own rooms, or exporting the created ranking list to Spotify.
