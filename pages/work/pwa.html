This application is a server side rendered app. It also contains some functionalities like a Service Worker and build scripts with Gulp. During this assignment one of the must haves was that the application would become a progressive web application. I partially succeeded in this assignment. The application can be used on all possible devices, it is a fast application and can also be used offline. However, this is not a large application.

I used a football API for this application. The data that I fetched from the API was not that big that the performance of the application could be improved in large steps. The little things like compression via Express made almost no difference to the use of the application. And a clear caching strategy could not be developed either. That does not mean that I was fortunately able to learn a lot about these subjects.
