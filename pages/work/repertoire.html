Repertoire wants to solve problems within the music industry by addressing the entire flow from musical work, sound recordings, licensing and rights associated with payment flows. We made a storytelling webpage to explain what Repertoire wants to achieve. For this website we used D3 to create a 'treemap'. This was the starting point. Our first idea was to create a Prezi-like website, so the user can follow the line of the money. However, after some prototyping we found out that it would not be really easy to read if we made it that way.

So, we modified our first idea a bit. Every time the user arrives at a new section, the piece of text is supported by an animation. Partly to clarify the text, but also to make the difficult text enjoyable when the user is reading. With Lottie which is a library that parses Adobe After Effects animations exported as JSON with Bodymovin and renders them natively on the web, we were able to show the different animations.
