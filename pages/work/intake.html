For this project my goal was to (re-)design the user interface and user experience of this tool. This tool is intended for candidate minor student for the study Communication and Multimedia Design. With this tool the students take on a code challenge. Other contributors of this project already made the backend side and created a first prototype of the user interface.

There are a few things I learned after this project and what I can take with me to the next project I will be working on. First, I need to create more issues on Github. This makes it clearer to other participants what has changed in the project where I worked on. Second, I should have created a branch for each feature I developed so that it was clearer to other contributors as well. After all, it was a good learning process for me.
